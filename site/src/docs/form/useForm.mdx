<h1 id="useForm">useForm</h1>

useForm 为 Form 供出一系列的 api，其中包括表单的取值、设值、校验、重置等。

```typescript
const form = useForm("demo_form");
```

## 参数

### formName : string

---

表单的名称（即表单的唯一标识）。

**_ ⓘ 表单名称不可重复，否则会导致表单数据覆盖，useForm 所提供的 api 不可用等无法预知的问题。_**

## 返回值
